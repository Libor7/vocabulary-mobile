## How to add a new language

To add a new language to this application, create a new folder inside the `locales` directory.
The folder name must be a valid language tag (for example `es`, `fr`, `it`).

Inside this folder, create the same JSON files as in the existing language folders
(e.g. `content.json`, `words.json`) and provide all required translations.

After adding the new language, start the application using:

```bash
npm start
```

This will automatically regenerate the app/i18n/resources.generated.ts file. Do not edit this file manually.
